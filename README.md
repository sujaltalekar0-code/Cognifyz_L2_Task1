# 🍽️ Table Booking and Online Delivery Analysis

## 📌 Objective

The objective of this project is to analyze restaurant data based on table booking and online delivery services. The analysis helps in understanding customer service trends, restaurant ratings, and the impact of pricing on online delivery availability.

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

---

## 📊 Tasks Performed

### 1️⃣ Percentage Analysis

* Calculated the percentage of restaurants offering:

  * Table Booking
  * Online Delivery

### 2️⃣ Ratings Comparison

* Compared the average aggregate ratings of restaurants:

  * With table booking
  * Without table booking

### 3️⃣ Price Range Analysis

* Analyzed how online delivery availability changes across different price ranges.

### 4️⃣ Data Visualization

Created visualizations using bar charts and count plots to better understand:

* Table booking distribution
* Online delivery distribution
* Ratings comparison
* Online delivery availability by price range

---

## 📈 Key Insights

* Restaurants offering table booking generally tend to have higher ratings.
* Online delivery services are more common in specific price ranges.
* Visual analysis helped identify customer service trends among restaurants.
